# Luxuria Booking Widget

Modern, serene booking engine for luxury hotels with direct inventory access.

## Tech Stack
- **Framework:** Next.js 16 + TypeScript
- **Styling:** Tailwind CSS (beige/cream aesthetic)
- **Fonts:** Inter (body) + Crimson Pro (headings)
- **Deployment:** Vercel
- **API:** UJV Staging API (direct hotel inventory)
